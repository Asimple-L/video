# 日志与工具类使用规范

## 1. 日志规范

### 1.1 日志框架

- 使用 Log4j2（已排除 Spring Boot 默认的 Logging）
- 通过自定义 `LogUtil` 工具类封装 SLF4J

### 1.2 日志调用方式

```java
// info 级别
LogUtil.info(FilmService.class, "--> 新增影片：" + film.getName());

// error 级别
LogUtil.error(FilmService.class, "更新失败，id=" + id);

// 不带类名参数的 error
LogUtil.error("系统异常：" + e.getMessage());

// 带异常对象的 error（推荐，可打印堆栈）
LogUtil.error(FilmService.class, e.getMessage(), e);
```

### 1.3 日志格式

- 消息前缀统一加 `--> `
- 日志内容清晰描述操作和关键数据

### 1.4 禁止事项

- **禁止**使用 `System.out.println()` 输出日志
- **禁止**使用 `e.printStackTrace()` 打印异常

## 2. 核心工具类

### 2.1 ResponseReturnUtil — 响应构建

```java
// 成功响应
ResponseReturnUtil.returnSuccessWithData(data)
ResponseReturnUtil.returnSuccessWithMsg("消息")
ResponseReturnUtil.returnSuccessWithoutMsgAndData()

// 失败响应
ResponseReturnUtil.returnErrorWithMsg("消息")
ResponseReturnUtil.returnErrorWithData(data)

// 预定义常量
ResponseReturnUtil.LOGIN_FIRST        // "请先登录"
ResponseReturnUtil.ILLEGAL_VISIT      // "非法访问"
```

### 2.2 RequestUtil — 请求鉴权

```java
RequestUtil.isLogin(request)              // 是否已登录
RequestUtil.isNotAdminLogin(request)      // 是否非管理员
RequestUtil.isNotSelfLogin(request, id)   // 是否非本人
```

### 2.3 PageBean — 分页封装

```java
PageBean<T> pageBean = new PageBean<>();
pageBean.setPc(1);           // 当前页码
pageBean.setPs(10);          // 每页大小
pageBean.setTr(100);         // 总记录数
pageBean.setBeanList(list);  // 数据列表
pageBean.setUrl(url);        // 条件URL
pageBean.getTp();            // 总页数（自动计算）
```

### 2.4 Tools — 通用工具

```java
Tools.UUID()              // 生成32位大写UUID
Tools.str2Date(str)       // 字符串转日期（注意：有 e.printStackTrace 遗留问题）
```

### 2.5 IBaseDao — Mapper 通用接口

```java
public interface IBaseDao<T> {
    Integer add(T t);
    Integer delete(Object id);
    Integer edit(T t);
    T getById(Object id);
}
```

### 2.6 VideoKeyNameUtil — Redis 缓存 Key 常量

```java
VideoKeyNameUtil.FILM        // 影片缓存分区
VideoKeyNameUtil.USER        // 用户缓存分区
VideoKeyNameUtil.CATALOG     // 分类缓存分区
```

### 2.7 JSONUtil — JSON 转换

```java
// 将对象转换为 JSONObject
JSONObject json = JSONUtil.getJSON(obj);
```

### 2.8 MD5 / MD5Auth — 加密

```java
MD5.getMD5(str)          // MD5 加密
MD5Auth.authUser(user)   // 用户密码校验
```

### 2.9 PropertiesUtil — 属性文件读取

```java
PropertiesUtil.getValue("key")  // 读取 user.properties 中的值
```

## 3. 分页计算模式

Service 层手动计算分页偏移量：

```java
int pc = 1;   // 当前页码
int ps = 10;  // 每页大小

int offset = (pc - 1) * ps;

Map<String, Object> param = new HashMap<>(4);
param.put("start", offset);
param.put("count", ps);
```

## 4. 通用规范

- 集合初始化时指定预期容量：`new HashMap<>(4)`、`new ArrayList<>()`
- 字符串拼接使用 `StringBuilder`（**不**使用 `StringBuffer`）
- 统一使用 Fastjson（`com.alibaba.fastjson`），**不**混用其他 JSON 库
