# 统一响应格式规范

## 1. 响应结构

所有 Controller 接口统一返回 `JSONObject`（Fastjson），结构如下：

```json
{
    "code": "000000",
    "message": "操作成功",
    "data": { ... }
}
```

## 2. 响应码约定

| code | 含义 |
|------|------|
| `000000` | 成功 |
| `111111` | 失败 |

## 3. 构建工具类

使用 `com.asimple.util.ResponseReturnUtil` 构建响应：

### 3.1 成功响应

```java
// 成功 + 数据
return ResponseReturnUtil.returnSuccessWithData(data);
// 输出: {"code":"000000","data":{...}}

// 成功 + 消息
return ResponseReturnUtil.returnSuccessWithMsg("操作成功");
// 输出: {"code":"000000","message":"操作成功"}

// 成功 + 无消息无数据
return ResponseReturnUtil.returnSuccessWithoutMsgAndData();
// 输出: {"code":"000000"}
```

### 3.2 失败响应

```java
// 失败 + 消息
return ResponseReturnUtil.returnErrorWithMsg("错误信息");
// 输出: {"code":"111111","message":"错误信息"}

// 失败 + 数据
return ResponseReturnUtil.returnErrorWithData(data);
// 输出: {"code":"111111","data":{...}}
```

## 4. 预定义错误消息

```java
// 定义在 ResponseReturnUtil 中的常量
ResponseReturnUtil.LOGIN_FIRST       // "请先登录"
ResponseReturnUtil.ADMIN_LOGIN_FIRST // "请以管理员身份登录"
ResponseReturnUtil.ILLEGAL_VISIT     // "非法访问"
ResponseReturnUtil.SERVER_ERROR      // "服务器错误"
```

使用方式：

```java
if (!RequestUtil.isLogin(request)) {
    return ResponseReturnUtil.returnErrorWithMsg(ResponseReturnUtil.LOGIN_FIRST);
}
```

## 5. JSON 库

- 统一使用 Alibaba Fastjson（`com.alibaba.fastjson.JSONObject`）
- **不**使用 `net.sf.json-lib` 或 `org.json`
- 返回声明为 `Object`，实际返回 `JSONObject`

```java
@RequestMapping(value = "/test", method = RequestMethod.GET, produces = "application/json;charset=UTF-8")
public Object test() {
    return ResponseReturnUtil.returnSuccessWithMsg("成功");
}
```
